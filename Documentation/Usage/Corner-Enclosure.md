
# Creating a Corner Enclosure

This tutorial will guide you through creating a basic corner shower enclosure.

<br/>

## Step 1: Create the Enclosure

1. Open FreeCAD and switch to the **ShowerDesigner** workbench
2. Click the **Corner Enclosure** icon in the toolbar
3. A new corner enclosure object will appear in your document

<br/>

## Step 2: Adjust Dimensions

Select the enclosure in the tree view and modify the properties:

### Dimensions
- **Width**: Set to your desired width (e.g., 900mm)
- **Depth**: Set to your desired depth (e.g., 900mm)
- **Height**: Typically 2000mm for standard installations

### Glass Properties
- **Glass Thickness**: Common values are 6mm, 8mm, or 10mm
- **Glass Type**: Choose from Clear, Frosted, Tinted, or Pattern

<br/>

## Step 3: Configure Frame

- **Show Frame**: Enable to add frame profiles
- **Frame Width**: Set profile width (typically 20-30mm)

<br/>

## Step 4: Review and Adjust

- Rotate the 3D view to inspect all sides
- Make any necessary adjustments
- The model updates automatically

<br/>

## Step 5: Export

Once satisfied with your design:

1. Click the **Export** button in the toolbar
2. Choose your export format (DXF, STEP, etc.)
3. Save the file for fabrication

<br/>

## Tips

- Start with standard dimensions and adjust as needed
- Check local building codes for minimum enclosure sizes
- Consider door swing clearance when planning layout
- Use thicker glass (10mm) for frameless designs
