# ShowerDesigner

Parametric shower enclosure design workbench for FreeCAD.

<img width = '360' src = './Resources/Media/Header.webp' />

</div>

<br/>
<br/>

## Features

- **Parametric Design**: Create custom shower enclosures with adjustable dimensions
- **Multiple Configurations**: Support for corner, alcove, walk-in, and custom layouts
- **Glass Panel Management**: Configure glass panels with proper thickness and spacing
- **Hardware Integration**: Add hinges, handles, seals, and support bars
- **Manufacturing Ready**: Export designs with cut lists and assembly instructions

<br/>

## Installation

### Via FreeCAD Addon Manager (Recommended)
1. Open FreeCAD
2. Go to `Tools` â†’ `Addon Manager`
3. Search for "ShowerDesigner"
4. Click Install

### Manual Installation
1. Clone or download this repository
2. Copy the `ShowerDesigner` folder to your FreeCAD `Mod` directory:
   - **Linux**: `~/.local/share/FreeCAD/Mod/`
   - **Windows**: `C:\Users\YourUsername\AppData\Roaming\FreeCAD\Mod\`
   - **macOS**: `~/Library/Application Support/FreeCAD/Mod/`
3. Restart FreeCAD

<br/>

## Quick Start

1. Open FreeCAD and switch to the **ShowerDesigner** workbench
2. Create a new document
3. Use the toolbar to:
   - Insert a shower enclosure template
   - Adjust dimensions via the properties panel
   - Add glass panels and hardware
   - Generate manufacturing drawings
