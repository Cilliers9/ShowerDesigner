# Bevel Range — Shower Hinge Specifications

Source: Showers-Ex-Sliding-Catalogue.pdf (pages 3-7)

> Dimensions in mm. Verify against original PDF technical drawings.
> `**` = Adjustable Series.

---

## Code Pattern

`SDH-{series}{type}-{angle}{suffix}`

| Series | Material | Finish |
|--------|----------|--------|
| 201 | S/S 304 | Bright Polished |
| 281 | S/S 304 | Antique Brass |
| 291 | S/S 304 | Matte Black |
| 301 | Brass | Bright Chrome |
| 501 | S/S 304 | Bright Polished |
| 581 | S/S 304 | Antique Brass |
| 591 | S/S 304 | Matte Black |
| 701 | Brass | Bright Chrome |
| 711 | Brass | Satin Chrome |
| 791 | Brass | Matte Black |

- Type `1` = wall-to-glass, Type `2` = glass-to-glass, Type `4` = wall pivot, Type `8` = glass pivot
- Suffix `HP` = Half Plate, `E` = Equal Cut Out
