
# ShowerDesigner

Parametric shower enclosure design workbench for FreeCAD.

<img width = '300' src = '../Media/Header.webp' />


## What is ShowerDesigner?

ShowerDesigner is a FreeCAD workbench that enables architects, designers, and manufacturers to create parametric shower enclosure designs with precision and ease.

### Key Features

**Parametric Design**
- Fully adjustable dimensions
- Automatic constraint management
- Real-time preview updates

**Multiple Configurations**
- Corner enclosures
- Alcove installations
- Walk-in showers
- Custom layouts

**Professional Output**
- Manufacturing-ready drawings
- Cut lists with dimensions
- Assembly instructions
- Material specifications

**Hardware Integration**
- Extensive hardware library
- Automatic placement suggestions
- Custom hardware support


## Who is it for?

- **Architects & Designers**: Create accurate shower designs for bathroom projects
- **Manufacturers**: Generate production-ready specifications
- **Contractors**: Visualize installations and verify dimensions
- **DIY Enthusiasts**: Design custom shower enclosures


## Getting Started

1. Install via FreeCAD Addon Manager
2. Switch to the ShowerDesigner workbench
3. Choose a template (Corner, Alcove, or Walk-in)
4. Adjust dimensions in the properties panel
5. Add glass panels and hardware
6. Export drawings and specifications


## Requirements

- FreeCAD 0.20 or later
- Python 3.10+
- NumPy 2.4.0+
